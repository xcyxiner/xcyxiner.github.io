2.1 引擎

Egret Engine 是一款使用 TypeScript 编写的 HTML5 游戏引擎，包含渲染、声音、用户交互、资源管理等诸多功能

2.2 工具

工具说明
产品中心

推荐安装

工具名称 备注
Egret Wing 3 EgretWing3 是Egret团队基于VSCode开发的最新编辑器
Res Depot Res Depot 是 Egret 游戏的可视化资源管理工具，能够轻松高效地管理海量游戏素材和配置文件资源。
Texture Merger Texture Merger 是一款纹理集打包工具，能批量将零散碎图合并为大图纹理集，提升游戏性能。
Egret UI Editor Egret UI Editor 是一款针对白鹭项目的可视化 UI 编辑器。

可后续再安装或者不安装

工具名称 备注
Lakeshore Lakeshore 是一款免费无需编程的游戏创作工具，通过Lakeshore可以轻松实现自己的游戏创作梦想。
Egret Inspector Egret Inspector 是一款供 Chrome 开发者使用的插件，能够帮助开发者可视化地调试 Egret 项目。
Egret Feather Egret Feather 是一款粒子编辑器，各个参数的组合塑造千变万化的效果，为游戏添姿添彩。
DragonBones DragonBones 是一款开源免费的移动游戏骨骼动画解决方案。
Egret Native Egret Native 可以将白鹭引擎制作的游戏发布成原生项目，有完善的资源缓存策略和热更新机制。
Egret Pro Egret Pro 是一款以游戏设计师为中心的游戏创作工具

2.3 安装顺序

参考推荐安装的顺序
Egret Wing 3 > Res Depot > Texture Merger > Egret UI Editor
建议单个下载，如果中间网络出问题，建议在download目录中删除对应的文件名
